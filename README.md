校园集市项目简介
本校园集市项目是基于SpringBoot + Vue3 + Pinia + MyBatis 技术栈、遵循 SSM（Spring + SpringMVC + MyBatis）核心框架设计理念，以 Maven 为构建工具、在 IntelliJ IDEA 开发环境中完成开发与部署的一站式校园综合服务平台。项目聚焦校园场景下的学生核心需求，打造集社交、交易、沟通、资源共享于一体的轻量化生态体系，旨在解决校园内交友难、闲置物品流转慢、信息沟通不便捷、学习资源分散等痛点，为在校学生提供安全、高效、贴合校园生活的数字化服务。
核心技术架构
后端：基于 SpringBoot 快速构建微服务化后端体系，整合 SpringMVC 实现请求分发与控制、MyBatis 完成数据库交互，依托 SSM 框架的低耦合特性保障业务逻辑的可扩展性与稳定性；
前端：采用 Vue3 作为核心视图框架，通过 Pinia 实现轻量化状态管理，结合 Vue Router 完成页面路由跳转，打造响应式、交互友好的前端界面；
工程管理：使用 Maven 统一管理项目依赖，支持一键打包、编译与运行，适配 IDEA 开发环境的调试与部署流程，确保项目快速启动与稳定运行。
核心功能模块
1. 找搭子交友模块
为学生提供精准化校园社交入口，用户可发布交友需求、查看他人交友帖并发起互动，系统基于标签化匹配算法推荐潜在搭子，打破校园社交壁垒，满足学生结伴学习、活动、出行等社交需求。
2. 闲置物品出售模块
聚焦校园闲置资源循环利用，学生可发布闲置物品（书籍、电子产品、生活用品、学习资料等）信息，包含图文描述、价格等核心内容；支持物品分类检索、价格筛选、收藏等功能，买卖双方可直接查看物品详情，且可以通过站内邮箱联系沟通交易方式，不必暴露真实联系方式，为校园闲置交易提供安全、便捷的线上渠道，降低交易成本。
3. 站内邮箱沟通交流模块
内置轻量化站内邮箱系统，作为平台内安全的沟通渠道，支持用户间点对点消息发送、回复、已读 / 未读状态标识，以及消息通知提醒功能。无论是闲置交易的细节沟通，还是搭子交友的前期交流，均无需跳转外部社交工具，保障沟通隐私与校园场景的适配性。
4. 共享资料资源模块
打造校园学习资源共享池，支持学生上传、下载各类学习资料（课件、真题、复习笔记等），按资料类型分类管理；实现校园内学习资源的高效流转与复用，降低学生获取学习资料的成本。
项目特点
轻量化部署：基于 SpringBoot 的自动配置特性，项目可一键启动，无需复杂的环境配置，适配校园服务器或个人开发环境的快速部署；
校园场景适配：所有功能均围绕学生日常需求设计，规避通用社交 / 交易平台的冗余功能，贴合校园生活节奏与使用习惯；
技术体系主流：采用 Vue3 + Pinia + SpringBoot 等当下主流技术栈，代码结构清晰、易维护，便于后续功能迭代与扩展；
数据安全可靠：基于 SSM 框架的权限控制与数据校验机制，保障用户信息、交易记录、资源文件等数据的安全性与完整性。
本项目可直接在 IDEA 中通过 Maven 指令（mvn spring-boot:run）启动，后端接口与前端页面无缝对接，实现 “开箱即用” 的校园集市服务，为高校学生构建了一个集社交、交易、沟通、资源共享于一体的综合性校园数字化平台。
